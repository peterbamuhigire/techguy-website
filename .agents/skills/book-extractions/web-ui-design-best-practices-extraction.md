# Bank / Cao - Web UI Design Best Practices

**Source:** Chris Bank and Jerry Cao, *Web UI Design Best Practices*.

## Use this for

- UI planning grounded in user goals
- Visual hierarchy
- Pattern selection
- Design collaboration

## High-signal insights

1. UI is the visible working surface of UX, not a cosmetic layer after UX is finished.
2. Start with top tasks and user scenarios before selecting interface patterns.
3. Use visual hierarchy to direct attention before adding more interface elements.
4. Committees dilute design decisions; structured critique and design-studio style exploration work better.
