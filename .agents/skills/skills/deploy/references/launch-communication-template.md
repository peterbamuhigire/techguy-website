# Launch Communication Template

Use this template once production verification is complete.

## Client Launch Confirmation

Subject:
`[Project Name] is now live`

Body:

```text
Hello [Client Name],

We are writing to confirm that [Project Name] has now been launched on the live domain:
[Live URL]

The following launch checks have been completed:
- core page access
- navigation verification
- form submission verification
- analytics verification
- search-engine-facing files verification

Known items, if any:
- [Item]

Next scheduled reviews:
- 7-day review: [date]
- 30-day review: [date]

If you notice anything unexpected, please send it to [contact] and include the page URL, device, browser, and a screenshot where possible.

Kind regards,
[Name]
```
