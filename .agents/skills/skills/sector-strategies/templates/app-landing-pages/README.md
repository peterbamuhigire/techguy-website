# App Landing Pages Sector Templates

Design strategy for mobile app, SaaS, and software product landing pages.

## Files

| File | Purpose |
|------|---------|
| **design-tokens.md** | Reference palettes for 5 app categories, typography, responsive system |
| **conversion-strategies.md** | Conversion frameworks (AIDA, PAS, FAB), A/B testing strategies |
| **implementation-guide.md** | Step-by-step build guide |
| **app-branding.md** | Color psychology, CTA color strategy, typography by app type, screenshot strategy |
| **app-conversion.md** | Download funnel, hero design, CTA strategy, pricing pages, trust architecture |
| **app-type-patterns.md** | 7 types (SaaS, consumer, dev tools, health, finance, gaming, education) |

## Usage

1. Read **app-branding.md** to understand app landing page design psychology
2. Read **app-conversion.md** for download funnel and conversion patterns
3. Read **app-type-patterns.md** to find your app type
4. Review **conversion-strategies.md** for framework selection
5. Run the design-system skill

App landing pages have one job: convert visitors into downloads/signups. Every design decision must serve conversion. 70%+ of traffic is mobile.
