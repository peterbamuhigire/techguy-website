# Corporate & Consulting Sector Templates

Design strategy for corporate, consulting, and enterprise websites.

## Files

| File | Purpose |
|------|---------|
| **sector-brief-template.md** | Customisation questionnaire (REQUIRED) |
| **design-tokens.md** | Reference palette, typography, spacing (DO NOT USE AS-IS) |
| **component-patterns.md** | HTML/Tailwind component examples |
| **corporate-branding.md** | Color psychology, typography by firm type, competitive differentiation, brand voice |
| **client-conversion.md** | B2B funnel, trust architecture, case study design, CTA hierarchy |
| **corporate-type-patterns.md** | 6 firm types (strategy, tech, financial, creative, legal, boutique) with design systems |

## Usage

1. Read **corporate-branding.md** to understand corporate design psychology
2. Read **client-conversion.md** for trust and conversion patterns
3. Read **corporate-type-patterns.md** to find your firm type
4. Copy **sector-brief-template.md** to `docs/sector-brief.md` and customise
5. Run the design-system skill

Every corporate site must look different. These are starting points, not final designs.
