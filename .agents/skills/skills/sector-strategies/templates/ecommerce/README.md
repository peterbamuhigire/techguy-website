# E-commerce Sector Templates

Design strategy for online stores, product-based businesses, and shopping experiences.

## Files

| File | Purpose |
|------|---------|
| **sector-brief-template.md** | Customisation questionnaire (REQUIRED) |
| **design-tokens.md** | Reference palette, typography, spacing (DO NOT USE AS-IS) |
| **component-patterns.md** | HTML/Tailwind component examples |
| **ecommerce-branding.md** | Color psychology by store type, typography, photography, brand voice |
| **shopping-conversion.md** | Shopping funnel, product page design, trust architecture, checkout optimisation |
| **store-type-patterns.md** | 7 types (luxury, fashion, food, electronics, handmade, beauty, kids) |

## Usage

1. Read **ecommerce-branding.md** to understand e-commerce design psychology
2. Read **shopping-conversion.md** for product page and checkout conversion patterns
3. Read **store-type-patterns.md** to find your store type
4. Copy **sector-brief-template.md** to `docs/sector-brief.md` and customise
5. Run the design-system skill

Cart abandonment is 70%. Every design decision must reduce friction and build purchase confidence.
