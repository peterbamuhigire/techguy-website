# Store Type Design Patterns

Each store type has distinct buyer psychology and design needs.

## Luxury / Premium

- **Palette:** Black + gold, or deep navy + cream. Minimal, high-contrast, breathing space.
- **Typography:** Elegant serif headers + thin sans body. Generous letter-spacing.
- **Layout:** Sparse, editorial. Large product images, minimal text, premium feel.
- **Key elements:** Full-screen product imagery, aspirational lifestyle shots, limited editions, VIP access CTA
- **Unique:** Exclusive collection launches, concierge chat, personalisation options, gift wrapping

## Fashion / Apparel

- **Palette:** Brand-driven. Often neutral base with bold seasonal accent.
- **Typography:** Bold, trend-forward sans. Changes with seasons/collections.
- **Layout:** Lookbook-style with shop-the-look functionality.
- **Key elements:** Size guide, model measurements, fabric details, outfit builder, wishlist, style quiz
- **Unique:** Virtual try-on, style quiz recommendations, outfit of the day, influencer collabs

## Food / Grocery

- **Palette:** Fresh greens, warm reds, earthy tones. Appetite-stimulating.
- **Typography:** Warm, friendly. Rounded sans or hand-crafted feel.
- **Layout:** Category-focused grid, recipe integration, subscription options.
- **Key elements:** Freshness indicators, dietary filters (vegan, gluten-free), recipe links, subscription/bundle options
- **Unique:** Meal planner, recipe integration, subscription box builder, dietary preference filter

## Electronics / Tech

- **Palette:** Dark theme or clean white + blue accent. Technical, modern.
- **Typography:** Geometric sans. Clean, precise, specification-friendly.
- **Layout:** Spec-heavy with comparison tools, detailed feature breakdown.
- **Key elements:** Spec comparison tables, compatibility checker, tech support links, warranty info, unboxing videos
- **Unique:** Product comparison tool, compatibility checker, spec sheet download, customer Q&A section

## Handmade / Artisan

- **Palette:** Earth tones, warm neutrals + brand accent. Authentic, tactile.
- **Typography:** Distinctive serif or hand-drawn. Reflects craftsmanship.
- **Layout:** Story-driven, maker profiles, process documentation.
- **Key elements:** Maker story, process photos, materials sourcing, custom order form, limited editions
- **Unique:** "Meet the maker" profiles, behind-the-scenes process gallery, custom order builder

## Beauty / Cosmetics

- **Palette:** Blush + gold or black + rose gold. Elegant, sensory.
- **Typography:** Refined sans or modern serif. Clean, beauty-editorial feel.
- **Layout:** Swatch-heavy, shade finders, routine builders.
- **Key elements:** Shade finder/matcher, ingredient list, routine builder, before/after, tutorial links
- **Unique:** Skin type quiz, shade matching tool, routine recommendation engine, ingredient glossary

## Kids / Family

- **Palette:** Bright, joyful. Primary colours + pastels. High energy.
- **Typography:** Rounded, friendly. Readable for parents scanning quickly.
- **Layout:** Age-grouped, safety-certified, gift guide integration.
- **Key elements:** Age-appropriate filtering, safety certifications, gift guides, bundle builders, registry integration
- **Unique:** Age-based product finder, safety certification display, gift registry, educational value indicators

## Branding Personality Framework

| Trait | Spectrum | Design Impact |
|-------|----------|---------------|
| **Price Point** | Budget ←→ Premium | Layout density, white space, imagery quality |
| **Audience** | Broad ←→ Niche | Category structure, personalisation depth |
| **Purchase Type** | Impulse ←→ Considered | CTA urgency, information depth, comparison tools |
| **Brand** | Marketplace ←→ Own brand | Branding prominence, storytelling depth |
| **Sustainability** | Standard ←→ Eco-first | Green messaging, certifications, packaging info |
