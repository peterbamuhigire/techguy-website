# Education Sector Templates

Design resources for school and educational institution websites. Covers public, private, charter, international, higher education, religious, and vocational schools.

## Files in This Directory

### Core Templates

| File | Purpose | Status |
|------|---------|--------|
| **sector-brief-template.md** | Customization questionnaire — COPY to `docs/sector-brief.md` and fill in | REQUIRED |
| **design-tokens.md** | Reference color palette, typography scale, component specs | Reference only |
| **component-patterns.md** | Basic HTML/Tailwind component examples | Pattern reference |
| **component-patterns-full.md** | Complete component library with reference site inspiration | Pattern reference |
| **IMPLEMENTATION-GUIDE.md** | Full Astro + Tailwind implementation guide for Uganda schools | Implementation |

### Reference Guides (NEW)

| File | Purpose |
|------|---------|
| **school-branding.md** | Color psychology for schools, typography pairing by school type, logo/mascot guidelines, photography direction, brand voice, competitive differentiation |
| **enrollment-design.md** | Enrollment conversion funnel, parent trust architecture (3 tiers), CTA hierarchy, fee structure design, testimonial patterns, form UX, virtual tour design |
| **school-type-patterns.md** | Design patterns for 7 school types: public, private, charter, international, higher ed, religious, vocational. Branding personality framework (6 traits) |

## How to Use

1. Read **school-type-patterns.md** to identify your school type
2. Read **school-branding.md** for color and typography guidance
3. Copy **sector-brief-template.md** to `docs/sector-brief.md` and answer ALL questions
4. Read **enrollment-design.md** for conversion best practices
5. Run the `design-system` skill (reads your sector-brief + design-tokens.md)
6. Reference **component-patterns-full.md** during page-builder phase
