# School Type Design Patterns

No two schools should look the same — even within the same type. Each school type has distinct trust signals, audience expectations, and design psychology. Choose your type, then customize heavily via your sector-brief.

## Public / Government Schools

**Audience:** Local community families, budget-conscious parents
**Trust signals:** Government affiliation, inspection results, community roots, exam pass rates

### Design Direction

- **Palette:** Community-warm colors. Avoid cold corporate. Earth tones + one bright accent.
- **Typography:** Clean, accessible sans-serif (Nunito, Source Sans Pro). Nothing flashy.
- **Photography:** Community events, diverse student body, local context, outdoor spaces.
- **Tone:** Welcoming, inclusive, community-proud. "Your neighbourhood school."

### Key Design Elements

| Element | Implementation |
|---------|---------------|
| **Government badges** | Education ministry logo, inspection rating badge in header |
| **Catchment area map** | Show which areas the school serves |
| **Free/subsidized signals** | "Government-funded" badge, "No tuition fees" prominent |
| **Community gallery** | Sports days, parent meetings, cultural events |
| **Simple navigation** | 5-6 top-level items max — parents want simplicity |
| **Multilingual support** | Local language + English for diverse communities |

### Unique Components
- School meal programme information
- Transport/bus routes
- After-school clubs list
- PTA/parent committee section
- Government curriculum overview

## Private / Independent Schools

**Audience:** Affluent parents, expatriates, aspirational families
**Trust signals:** Academic results, university placements, facilities quality, faculty credentials

### Design Direction

- **Palette:** Premium, distinctive. Deep navy + gold, emerald + cream, or burgundy + slate. Never default blue.
- **Typography:** Serif headlines (Playfair Display, Cormorant) + clean sans body. Premium feel.
- **Photography:** High-quality professional shots. Polished campus, small class sizes, modern facilities.
- **Tone:** Confident, aspirational, exclusive but welcoming. "Excellence in education."

### Key Design Elements

| Element | Implementation |
|---------|---------------|
| **Results showcase** | University acceptance rates, exam scores, percentile rankings |
| **Facilities tour** | Interactive virtual tour with high-quality imagery |
| **Small class sizes** | "12:1 student-teacher ratio" stat prominently displayed |
| **Faculty credentials** | Degrees, international experience, publications |
| **Fee transparency** | Clear fee structure with financial aid information |
| **Alumni network** | Notable alumni, career outcomes, reunion events |

### Unique Components
- Scholarship calculator / financial aid estimator
- Compare programmes (IB vs Cambridge vs National)
- Student achievement timeline (awards, competitions)
- Parent portal preview (show the value)
- Virtual open day booking system

## Charter / Community Schools

**Audience:** Mission-aligned families, education innovators, community members
**Trust signals:** Mission clarity, community involvement, innovation track record, outcomes

### Design Direction

- **Palette:** Mission-aligned. Environmental school = green + earth. Arts = bold + vibrant. STEM = teal + electric.
- **Typography:** Modern, approachable. Geometric sans (Outfit, Plus Jakarta Sans) for progressive feel.
- **Photography:** Students in action — project-based learning, hands-on activities, community projects.
- **Tone:** Mission-driven, innovative, community-centred. "Education reimagined."

### Key Design Elements

| Element | Implementation |
|---------|---------------|
| **Mission statement** | Prominent in hero, not buried in About page |
| **Innovation showcase** | Unique programmes, teaching methods, special projects |
| **Community involvement** | Volunteer opportunities, community partners, local impact |
| **Lottery/enrollment info** | Clear process for charter school lottery system |
| **Board transparency** | Board members, meeting minutes, governance |
| **Impact metrics** | Specific outcomes tied to mission (not just test scores) |

### Unique Components
- Mission alignment quiz ("Is this school right for your family?")
- Project showcase gallery (student work)
- Community partner logos
- Board meeting calendar
- Founder's story section

## International Schools

**Audience:** Expatriate families, globally mobile professionals, local elite
**Trust signals:** International accreditation (IB, CIS, WASC), global university placements, diversity

### Design Direction

- **Palette:** Globally sophisticated. Avoid single-nation colours. Teal + gold, deep blue + coral, or sage + navy.
- **Typography:** International-friendly fonts with broad character support. Plus Jakarta Sans, Inter (for body), or Noto Sans.
- **Photography:** Diverse student body, international events, modern facilities, global connections.
- **Tone:** Worldly, inclusive, excellence-focused. "A world of possibilities."

### Key Design Elements

| Element | Implementation |
|---------|---------------|
| **Accreditation badges** | IB World School, CIS, Cambridge, WASC logos prominently displayed |
| **Language support** | Site available in 2-4 languages, language switcher in header |
| **University placements** | World map showing where graduates go |
| **Diversity stats** | "45 nationalities represented" — prominent stat |
| **Curriculum comparison** | IB vs Cambridge vs American vs National — clear table |
| **Relocation support** | Transition programmes, housing help, community orientation |

### Unique Components
- World map of student nationalities (interactive)
- Curriculum pathway diagram (PYP → MYP → DP)
- Global university acceptance board
- Cultural events calendar (international days, heritage celebrations)
- Transition/new family welcome guide
- Embassy/diplomatic community resources

## Higher Education / University

**Audience:** Prospective students (17-25), parents, researchers, employers, alumni
**Trust signals:** Rankings, research output, employer partnerships, alumni career outcomes

### Design Direction

- **Palette:** Heritage colours often mandatory (use institutional colours). Add modern accent for digital.
- **Typography:** Heritage serif for prestige (EB Garamond, Cormorant Garamond) + modern sans for clarity.
- **Photography:** Campus life, research labs, graduation, student organisations, events.
- **Tone:** Authoritative yet inviting. "Where ambition meets opportunity."

### Key Design Elements

| Element | Implementation |
|---------|---------------|
| **Rankings/stats hero** | National ranking, student satisfaction %, research funding |
| **Programme finder** | Searchable/filterable database of all programmes |
| **Research showcase** | Featured publications, labs, faculty research profiles |
| **Campus experience** | Virtual tour, student life videos, club listings |
| **Application portal** | Multi-step application wizard with document upload |
| **Alumni success** | Career outcomes 5 years post-graduation, notable alumni |

### Unique Components
- Programme comparison tool (side-by-side)
- Research publication feed
- Campus map (interactive with building info)
- Student union / clubs directory
- Academic calendar with exam periods
- Employer partnership showcase

## Religious / Faith-Based Schools

**Audience:** Faith-aligned families, community members, values-driven parents
**Trust signals:** Religious affiliation, values education, moral development, faith community

### Design Direction

- **Palette:** Reverent but not sombre. Gold + deep blue (Catholic), green + gold (Islamic), burgundy + cream (Anglican). Warm, welcoming.
- **Typography:** Serif headers for tradition (Cormorant, EB Garamond, Lora). Warm sans for body.
- **Photography:** Chapel/mosque/prayer spaces, community worship, values-in-action, character development.
- **Tone:** Faith-centred, nurturing, community-strong. "Faith, knowledge, and service."

### Key Design Elements

| Element | Implementation |
|---------|---------------|
| **Faith identity** | Religious symbol/logo alongside school crest |
| **Values framework** | Core values displayed prominently (not just listed) |
| **Chaplaincy/faith life** | Dedicated section for spiritual development |
| **Character education** | How faith informs character development |
| **Community service** | Outreach programmes, charity work, mission trips |
| **Parent faith community** | Prayer groups, parent fellowship, family events |

### Unique Components
- Daily/weekly scripture or reflection
- Chapel/worship schedule
- Values-in-action student stories
- Faith milestones (first communion, confirmation, etc.)
- Mission/outreach photo gallery
- Parent prayer group signup

## Vocational / Technical Schools

**Audience:** Career-focused students, parents seeking practical pathways, industry employers
**Trust signals:** Employment rates, industry partnerships, certifications, practical facilities

### Design Direction

- **Palette:** Industrial-modern. Slate + electric blue, charcoal + safety orange, steel + green. Bold, functional.
- **Typography:** Modern sans (Rubik, DM Sans, Outfit). No-frills, professional, clear.
- **Photography:** Workshops, labs, hands-on training, industry placements, real-world equipment.
- **Tone:** Practical, career-focused, opportunity-driven. "Skills that work."

### Key Design Elements

| Element | Implementation |
|---------|---------------|
| **Employment stats** | "92% employed within 6 months" — hero-level prominence |
| **Industry partners** | Employer logos, partnership descriptions, placement programmes |
| **Certifications** | Industry-recognised qualifications and accreditations |
| **Workshop showcase** | Photo/video tour of practical facilities |
| **Student projects** | Portfolio of real-world projects completed |
| **Career pathways** | Visual diagram: Course → Certification → Employment |

### Unique Components
- Skills/trade programme cards with duration + certification
- Industry partnership grid or manually controlled logo reel
- Student project portfolio gallery
- Apprenticeship/placement finder
- Equipment and facilities showcase
- Graduate employment board (where alumni work)

## Branding Personality Framework

Every school has a unique personality within its type. Rate these 6 traits on a spectrum to define YOUR school's visual identity:

| Trait | Spectrum | Design Impact |
|-------|----------|---------------|
| **Formality** | Casual ←→ Formal | Typography weight, colour saturation, white space |
| **Tradition** | Modern ←→ Heritage | Serif vs sans-serif, colour warmth, imagery style |
| **Exclusivity** | Community ←→ Exclusive | Tone of voice, photography style, CTA language |
| **Energy** | Calm ←→ Dynamic | Animation, colour vibrancy, layout density |
| **Innovation** | Traditional ←→ Forward-looking | Tech showcase, modern UI patterns, interactivity |
| **Warmth** | Professional ←→ Nurturing | Colour temperature, photography intimacy, copy tone |

**Example mapping:**
- A traditional private school: Formal + Heritage + Exclusive + Calm + Traditional + Professional
- A progressive charter school: Casual + Modern + Community + Dynamic + Forward-looking + Nurturing
- These two schools should look COMPLETELY DIFFERENT even though they're both "education"

## Multi-Language Considerations

All school types in East Africa should consider multi-language support:

- **URL structure**: `/en/admissions/`, `/fr/admissions/`, `/sw/admissions/`
- **Content expansion**: French ~30% longer, Kiswahili ~20% longer than English
- **Curriculum terms**: May differ between languages (use official educational terminology)
- **Form labels**: Translate completely (not partially)
- **Date formats**: Consistent per language (British English for East African English)
