# Hobbyist/Creator Sector Templates

Design strategy for content creators, hobbyists, makers, and independent creators.

## Files

| File | Purpose |
|------|---------|
| **sector-brief-template.md** | Customisation questionnaire (REQUIRED) |
| **design-tokens.md** | Reference palette, typography, spacing (DO NOT USE AS-IS) |
| **component-patterns.md** | HTML/Tailwind component examples |
| **creator-branding.md** | Color psychology, typography by 7 creator types, photography, brand voice, logo types |
| **engagement-design.md** | Creator funnel, portfolio showcase, subscribe strategy, community building, monetisation |
| **creator-type-patterns.md** | 7 types (artist, developer, writer, photographer, YouTuber, craftsperson, educator) |

## Usage

1. Read **creator-branding.md** to understand creator brand design psychology
2. Read **engagement-design.md** for engagement, subscribe, and community patterns
3. Read **creator-type-patterns.md** to find your creator type
4. Copy **sector-brief-template.md** to `docs/sector-brief.md` and customise
5. Run the design-system skill

Creator sites sell personality and authenticity. Your website is your "home base" — it must feel distinctly like YOU, not a template.
