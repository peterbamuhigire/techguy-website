# Creator Type Design Patterns

Each creator type has distinct audience expectations and engagement patterns.

## Digital Artist / Illustrator

- **Palette:** Bold, saturated. Reflects art style. Often dark theme with vibrant accent.
- **Typography:** Display or hand-drawn headers + clean sans body. Expressive, artistic.
- **Layout:** Gallery-dominant, minimal text, full-bleed images, art-first.
- **Key elements:** Portfolio grid (masonry), commission info, process GIFs/videos, prints shop, style showcase
- **Unique:** Art style filter (by medium, theme, client), commission status indicator, print shop integration, speed-paint videos (YouTube embed)

## Developer / Tech Creator

- **Palette:** Dark mode default + electric accent (cyan, green, purple). Terminal-inspired.
- **Typography:** Monospace display (JetBrains Mono, Fira Code) + clean sans body. Technical, precise.
- **Layout:** Clean, structured, code-block friendly, data-rich.
- **Key elements:** Project cards (with tech stack badges), GitHub activity, blog with code snippets, open-source contributions, speaking engagements
- **Unique:** Interactive code demos, GitHub contribution graph, tech stack visualiser, terminal-style hero, live project links

## Writer / Blogger

- **Palette:** Warm, literary. Cream/off-white + rich accent (burgundy, forest, navy). Easy on eyes.
- **Typography:** Elegant serif headers (Cormorant, Playfair) + readable serif body (Lora, Crimson Text). 18px+ body.
- **Layout:** Content-first, generous margins, long-form optimised, minimal distraction.
- **Key elements:** Blog with categories/tags, book/publication list, writing samples, newsletter signup, reading time estimates
- **Unique:** Reading progress bar, estimated reading time, "also by me" section, publication timeline, writing process page

## Photographer / Videographer

- **Palette:** Minimal. Black/white/one accent. Let images dominate completely.
- **Typography:** Thin, minimal sans (Outfit 300, DM Sans 300). Never competes with images.
- **Layout:** Image-dominant, full-bleed galleries, minimal chrome, immersive viewing.
- **Key elements:** Gallery by category/project, before/after comparison modules, equipment list, booking form, location availability
- **Unique:** Fullscreen gallery mode, lazy-loaded high-res images, EXIF data display (optional), client gallery portal, booking calendar

## YouTuber / Content Creator

- **Palette:** Bold, energetic. Matches YouTube channel branding. High contrast.
- **Typography:** Bold, modern sans (Plus Jakarta Sans, Outfit 700). Energetic, thumbnail-inspired.
- **Layout:** Video-grid focused, merch integration, social feed, community links.
- **Key elements:** Latest videos feed (YouTube API or manual), merch shop, about/story, collaboration info, media kit
- **Unique:** Video embed grid, subscriber counter, merch store integration, sponsor/collaboration page, community Discord link

## Craftsperson / Maker

- **Palette:** Warm, natural. Earth tones, wood textures, handmade feel. Authentic.
- **Typography:** Slab serif or hand-drawn display + warm sans body. Tactile, artisanal.
- **Layout:** Large images showing detail/texture, process documentation, story-driven.
- **Key elements:** Product gallery with detail shots, process/making-of photos, materials sourcing, custom order form, workshop/class schedule
- **Unique:** Process timeline (raw → finished), material sourcing story, custom order builder, workshop booking, craft fair calendar

## Educator / Course Creator

- **Palette:** Accessible, trustworthy. Blue/green base + warm accent. Clear, organised.
- **Typography:** Friendly sans headers (Nunito Sans, DM Sans) + highly readable body. 18px minimum.
- **Layout:** Structured, curriculum-like, progress-oriented, resource-rich.
- **Key elements:** Course catalog, free resources, student testimonials, teaching philosophy, blog/articles, webinar schedule
- **Unique:** Course curriculum preview, student success stories, free mini-course funnel, certification badges, learning path visualisation

## Branding Personality Framework

| Trait | Spectrum | Design Impact |
|-------|----------|---------------|
| **Formality** | Casual ←→ Professional | Typography weight, photography style, voice tone |
| **Visibility** | Behind camera ←→ Front of camera | Hero image choice, about page prominence |
| **Niche** | Broad ←→ Hyper-specific | Content depth, audience targeting language |
| **Monetisation** | Free/ad-supported ←→ Premium/paid | Paywall design, pricing visibility, CTA language |
| **Community** | Solo creator ←→ Community leader | Discord/forum prominence, membership emphasis |
