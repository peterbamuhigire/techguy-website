# Personal & Portfolio Sector Templates

Design strategy for personal websites, portfolios, and individual brand sites.

## Files

| File | Purpose |
|------|---------|
| **sector-brief-template.md** | Customisation questionnaire (REQUIRED) |
| **design-tokens.md** | Reference palette, typography, spacing (DO NOT USE AS-IS) |
| **component-patterns.md** | HTML/Tailwind component examples |
| **portfolio-branding.md** | Color psychology, typography by creator type, photography, brand voice |
| **portfolio-conversion.md** | Portfolio funnel, project showcase design, social proof, CTA hierarchy |
| **portfolio-type-patterns.md** | 7 types (designer, developer, writer, photographer, illustrator, consultant, multi-disciplinary) |

## Usage

1. Read **portfolio-branding.md** to understand personal brand design psychology
2. Read **portfolio-conversion.md** for portfolio showcase and conversion patterns
3. Read **portfolio-type-patterns.md** to find your creator/professional type
4. Copy **sector-brief-template.md** to `docs/sector-brief.md` and customise
5. Run the design-system skill

Every personal site must look distinctly like YOU. These are starting points, not final designs.
