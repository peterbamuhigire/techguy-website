# Portfolio Type Design Patterns

Every creator type has distinct audience expectations and showcase needs.

## Designer (UI/UX/Product/Graphic)

- **Palette:** Bold or minimal — your portfolio IS the proof. Black + vibrant accent or clean white + colour.
- **Typography:** Display font that shows typographic taste. Never default system fonts.
- **Layout:** Project-image-heavy, large thumbnails, hover interactions, immersive case studies.
- **Key elements:** Process documentation (wireframes, iterations), before/after visuals, design system showcase, tool proficiency
- **Unique:** Interactive prototype embeds, Figma/design tool integration, design principle statements

## Developer / Engineer

- **Palette:** Dark theme or clean light. Accent colour for interactive elements.
- **Typography:** Monospace accent (JetBrains Mono) for code, sans for content.
- **Layout:** Clean, well-structured (demonstrates engineering thinking). Code snippets, architecture diagrams.
- **Key elements:** GitHub contribution graph, tech stack icons, live project links, code quality metrics, open-source contributions
- **Unique:** Terminal-style hero animation, live code demos, API documentation showcase

## Writer / Editor / Content Creator

- **Palette:** Warm, literary. Cream backgrounds, serif-forward, editorial feel.
- **Typography:** Serif-dominant (Cormorant, Literata, Source Serif). Reading-optimised layouts.
- **Layout:** Text-forward, magazine/editorial style, generous margins and line height.
- **Key elements:** Published work samples, publication logos, writing categories, word count/output metrics
- **Unique:** Reading-time estimates, publication filtering, excerpt previews, newsletter signup

## Photographer

- **Palette:** Minimal — black, white, or neutral. Let images be the colour.
- **Typography:** Thin, minimal sans (Jost, DM Sans light). Never compete with images.
- **Layout:** Full-bleed galleries, minimal UI, large images, masonry/grid layouts.
- **Key elements:** Gallery categories (portrait, landscape, event, commercial), EXIF data option, print ordering CTA
- **Unique:** Full-screen lightbox, image comparison module, project story captions, client gallery access

## Illustrator / Artist

- **Palette:** Reflects artistic style — bold and colourful, or muted and textural.
- **Typography:** Playful or hand-drawn feel. Reflects artistic personality.
- **Layout:** Gallery-forward with artistic flair. Process reveals, sketch-to-final progressions.
- **Key elements:** Medium/technique categorisation, commission information, process GIFs/videos, shop integration
- **Unique:** Custom cursor, animated illustrations, art print shop, commission request form

## Consultant / Coach / Speaker

- **Palette:** Professional but warm. Approachable authority — not cold corporate.
- **Typography:** Modern sans, confident weight. Readable and professional.
- **Layout:** Service-focused with testimonial prominence. Authority signals throughout.
- **Key elements:** Service packages, booking calendar, testimonials, speaking reel, podcast/media appearances
- **Unique:** Booking widget (Calendly-style), course/workshop listings, free resource downloads, podcast embed

## Multi-Disciplinary / Generalist

- **Palette:** Distinctive personal brand colour. Something that ties diverse work together.
- **Typography:** Versatile pairing that works across all project types.
- **Layout:** Filterable project grid, category tabs, diverse showcase.
- **Key elements:** Category filtering (design, development, writing, etc.), skill range visualisation, cross-discipline case studies
- **Unique:** Skill venn diagram, project type filtering, "jack of all trades" positioning statement

## Branding Personality Framework

| Trait | Spectrum | Design Impact |
|-------|----------|---------------|
| **Style** | Minimal ←→ Expressive | Animation amount, colour saturation, layout density |
| **Formality** | Casual ←→ Professional | Typography weight, photography style, voice tone |
| **Technical** | Creative ←→ Technical | Code visibility, process detail, tool prominence |
| **Personality** | Reserved ←→ Bold | Hero treatment, about section depth, personal photos |
| **Niche** | Generalist ←→ Specialist | Category breadth, expertise depth signals |
