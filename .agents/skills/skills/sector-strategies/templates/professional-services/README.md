# Professional Services Sector Templates

Design strategy for accounting, legal, architecture, engineering, IT, real estate, and marketing firms.

## Files

| File | Purpose |
|------|---------|
| **sector-brief-template.md** | Customisation questionnaire (REQUIRED) |
| **design-tokens.md** | Reference palette, typography, spacing (DO NOT USE AS-IS) |
| **component-patterns.md** | HTML/Tailwind component examples |
| **services-branding.md** | Color psychology, typography by 7 service types, brand voice |
| **client-acquisition.md** | Trust-first test, client funnel, service page design, testimonial strategy |
| **services-type-patterns.md** | 7 types (accounting, legal, architecture, engineering, IT, real estate, marketing) |

## Usage

1. Read **services-branding.md** to understand professional services design psychology
2. Read **client-acquisition.md** for client trust and conversion patterns
3. Read **services-type-patterns.md** to find your service type
4. Copy **sector-brief-template.md** to `docs/sector-brief.md` and customise
5. Run the design-system skill

82% of clients research providers online before making contact. The website must answer: "Are they expert? Can I trust them? Will they understand MY problem?"
