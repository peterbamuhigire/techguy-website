# Tour & Travel Sector Templates

Design resources for tour operator, travel agency, and tourism websites. Covers luxury safari, budget/backpacker, cultural immersion, eco-tourism, adventure/expedition, wellness/retreat, cruise, city breaks, and family travel.

## Files in This Directory

### Core Templates

| File | Purpose | Status |
|------|---------|--------|
| **sector-brief-template.md** | Customisation questionnaire — COPY to `docs/sector-brief.md` and fill in | REQUIRED |
| **design-tokens.md** | Reference color palette, typography scale, component specs, anti-homogeneity principle | Reference only |
| **component-patterns.md** | HTML/Tailwind component examples (destination cards, itinerary, booking CTA, gallery, hero, trust) | Pattern reference |
| **IMPLEMENTATION-GUIDE.md** | Full Astro + Tailwind implementation guide with booking flow, revenue optimisation, SEO, performance | Implementation |

### Reference Guides

| File | Purpose |
|------|---------|
| **travel-branding.md** | Color psychology for 9 tour types, typography pairing, photography direction (the #1 conversion driver), competitive differentiation, brand voice per audience, video guidelines |
| **booking-conversion.md** | 3-second test, booking funnel design, CTA hierarchy, urgency/scarcity patterns (ethical), trust architecture (3 tiers), pricing display rules, review design, search widget, gallery patterns, form UX, mobile considerations |
| **travel-type-patterns.md** | Design patterns for 9 travel types: luxury safari, budget/backpacker, cultural, eco-tourism, adventure, wellness, cruise, city breaks, family. Branding personality framework (6 traits) |

## How to Use

1. Read **travel-type-patterns.md** to identify your travel type
2. Read **travel-branding.md** for color and typography guidance
3. Copy **sector-brief-template.md** to `docs/sector-brief.md` and answer ALL questions
4. Read **booking-conversion.md** for conversion best practices
5. Run the `design-system` skill (reads your sector-brief + design-tokens.md)
6. Reference **component-patterns.md** during page-builder phase
7. Reference **IMPLEMENTATION-GUIDE.md** for booking flow and SEO
