# SEO Content Writing Reference

Rules for writing website and blog content that ranks in search engines while remaining natural, readable, and human. These rules apply to all page copy and blog articles.

Source: *Content and Copywriting Secrets* (GoldInk Books), *The One-Page Content Marketing Blueprint* (Prafull Sharma).

---

## Core Principle: Write for Humans First

Search spiders do not buy products, engage on social media, or become loyal customers. Write for human readers first, then optimise for search engines. Never reverse this priority.

---

## Keyword Density and Placement

### Hard Rules

| Rule | Standard |
|------|----------|
| Keyword density | Under 1% of total word count. Close to 2% reads as keyword stuffing. |
| First 50 words | Primary keyword must appear within the first 50 words, ideally mentioned 2-3 times naturally |
| Title (H1) | Primary keyword included, max 60 characters (Google truncates beyond this) |
| First H2 subheading | Primary keyword included |
| Meta description | Primary keyword included, max 150 characters (desktop), max 113 characters (mobile) |
| Body | Secondary/ancillary keywords spread naturally through the text |
| Conclusion | Primary keyword restated naturally |
| Internal links | Use keywords as anchor text for internal links to other pages/articles |

### What to Avoid

- Repeating a keyword so often it reads awkwardly
- Inserting keywords that break sentence flow
- Using the exact same keyword phrase repeatedly instead of natural variations
- Targeting broad, high-competition keywords when long-tail alternatives exist

## Search Intent Types

Every piece of content must match the searcher's intent. Three types:

| Intent | User Goal | Content Response | Examples |
|--------|-----------|-----------------|----------|
| **Informational** | Learn something | Educational articles, guides, how-tos | "What is ERP software?" |
| **Navigational** | Find a specific site/page | Branded content, about pages | "Maduuka login" |
| **Commercial** | Make a purchase decision | Service pages, comparisons, pricing, case studies | "Best ERP for SMEs Uganda" |

**Rule:** If content doesn't match the keyword's intent, it won't rank regardless of optimisation. An informational keyword needs an educational article, not a sales page.

## Featured Snippet Optimisation

More than half of Google searches now end without a click — users get their answer from the featured snippet. Optimise for snippets:

1. **Answer the question directly** in the first paragraph after the H2 heading (40-60 words)
2. **Use the question as the H2** — match common search queries exactly
3. **Follow with detail** — expand the answer in subsequent paragraphs
4. **Use structured formats** — numbered lists, tables, and definition-style paragraphs are favoured
5. **Target "what is", "how to", "why does"** question patterns

**Structure for snippet capture:**
```
## How to [keyword question]?

[Direct 40-60 word answer that could stand alone as a snippet]

[Expanded detail, examples, and nuance in subsequent paragraphs]
```

## Voice Search Optimisation

Two-thirds of users aged 25-49 use voice search daily. Voice queries differ from typed queries:

1. **Conversational phrasing** — voice searches use natural language ("Where can I find a good IT consultant near Kampala?" vs "IT consultant Kampala")
2. **Question-based** — most voice searches start with who, what, where, when, why, how
3. **Long-tail by nature** — voice queries are typically 5-9 words
4. **Local intent** — many voice searches include "near me" or location terms

**Implementation:**
- Include natural question phrases in H2/H3 headings
- Write opening answers in conversational, spoken language
- Target long-tail keyword phrases that mirror how people talk
- Ensure local SEO elements are present (location, region, service area)

## Long-Tail Keyword Strategy

70% of all search queries use long-tail keywords. These are easier to rank for and carry higher conversion intent.

| Broad (hard to rank) | Long-tail (easier, higher intent) |
|----------------------|----------------------------------|
| IT consultant | IT consultant for SMEs in Kampala |
| website design | website cost for small business Uganda |
| ERP system | ERP implementation mistakes African businesses |
| software developer | custom software developer Uganda ERP |

**Selection criteria:**
- Minimum 1,000 monthly searches for primary keywords
- Low to medium competition
- Clear commercial or informational intent
- Relevant to the client's services and geography

## On-Page SEO Checklist

For every page and blog article:

- [ ] Title tag includes primary keyword (max 60 characters)
- [ ] Meta description includes primary keyword + CTA (max 150 characters)
- [ ] Primary keyword in first 50 words
- [ ] Primary keyword in at least one H2 heading
- [ ] Keyword density under 1%
- [ ] At least 1 internal link to another page/article
- [ ] At least 1 external link to a credible source (per 500 words for articles)
- [ ] All images have descriptive alt text with keywords where natural
- [ ] URL slug includes primary keyword (short, hyphenated) and is localized for the page language
- [ ] Content matches the search intent of the target keyword
- [ ] Headings follow a logical hierarchy (H1 → H2 → H3)
- [ ] Page loads fast (no unnecessary scripts, optimised images)

## Content-to-Buyer-Journey Alignment

Content must match where the reader is in their buying journey:

| Journey Stage | Content Type | Keyword Focus | Goal |
|--------------|-------------|---------------|------|
| Awareness | Educational articles, problem-focused posts | Pain-point keywords | Help them understand their problem |
| Consideration | Comparisons, frameworks, advisory content | Solution keywords + modifiers | Position as knowledgeable guide |
| Decision | Case studies, portfolio, methodology deep-dives | Brand + review keywords | Prove capability, build confidence |

**Only 5% of website visitors are ready to buy.** The remaining 95% need content that moves them through these stages. A website with only decision-stage content ignores 95% of its potential audience.

## Search Traffic Distribution

Understanding where clicks go informs content ambition:

- Position 1: 32.5% of clicks
- Position 2: 17.6% of clicks
- Position 3: 11.4% of clicks
- First page total: 92% of all clicks

**Implication:** If you're not on page 1, you're effectively invisible. Target keywords where page 1 ranking is achievable, then expand to more competitive terms as domain authority grows.
