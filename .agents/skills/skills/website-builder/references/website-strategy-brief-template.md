# Website Strategy Brief Template

This is the canonical strategy document for premium website projects.
It exists to turn raw discovery into one execution-ready source of truth before
design work begins.

Use this after intake and discovery, and before `design-reference`,
`design-system`, `content-writing`, `page-builder`, or `seo` begin substantive work.

This template is shaped by:

- *Obviously Awesome* for positioning and competitive context
- *Building a StoryBrand* for message hierarchy and CTA logic
- *They Ask, You Answer* for buyer questions, transparency, and trust-led sales
- *Made to Stick* for clarity, specificity, credibility, and memorability

## Notes

- If this brief cannot justify a page, that page should not exist yet.
- If this brief cannot justify a proof requirement, that proof should not be implied in copy.
- If this brief is vague, downstream work will become generic.
