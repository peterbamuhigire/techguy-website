### Maduuka

**Client**: Proprietary product — Chwezi Digital Solutions

**Category**: SaaS / Business Management

**Short Summary**: A multi-tenant SaaS bookkeeping and business management platform designed for SMEs across Africa.

**Full Description**: Maduuka is a comprehensive business management platform built to serve the specific needs of small and medium-sized enterprises in African markets. The platform provides a complete suite of modules — Point of Sale, Inventory Management, Sales, Accounting, Pharmacy, Restaurant, and Asset Management — all within a single, integrated system. Designed as a multi-tenant SaaS platform, Maduuka enables franchise and multi-branch operations from one centralised system. The platform supports five languages (English, French, Arabic, Swahili, and Spanish), reflecting the diverse markets it serves. A companion Android mobile application provides offline-first capability, ensuring business operations continue even with unreliable internet connectivity — a critical requirement in many African markets.

**Key Highlights**:
- Multi-tenant architecture supporting franchise and multi-branch operations
- 7+ integrated modules: POS, Inventory, Sales, Accounting, Pharmacy, Restaurant, Asset Management
- 5 languages: English, French, Arabic, Swahili, Spanish
- Android mobile app with offline-first capability
- Serves businesses across East Africa
- From a single shop to multi-branch franchise — one platform

**Tech Stack**: PHP, MySQL, Kotlin/Jetpack Compose (Android), REST API

### Chwezi Server Manager

**Client**: Proprietary product — Chwezi Digital Solutions

**Category**: DevOps / Server Administration

**Short Summary**: A self-hosted server management tool providing complete server administration from a single dashboard.

**Full Description**: Chwezi Server Manager is a self-hosted web application that simplifies server administration tasks into a clean, unified interface. The tool handles domain management, SSL certificate provisioning, database operations, scheduled task management, activity monitoring, Git repository connections, and WordPress deployment — all from one dashboard. **Key Highlights**:

- Scheduled task management
