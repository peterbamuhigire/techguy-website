# Phase 2: Expert Positioning & Category Claim

## Score at Entry: 7.0/10 → Score at Exit: 7.4/10

Phase 1 fixed what was broken. Phase 2 fixes what is generic. The site currently reads as a capable individual with impressive credentials but no declared category, no named niche, and no signal that distinguishes it from any other ICT consultant with a website. This phase establishes the commercial identity that all subsequent copywriting and trust work must serve.

The single most important output of this phase is the **category claim** — a short, ownable statement of what Peter's practice *is* that no generic "IT consultant" descriptor can replicate. Everything else in Phase 2 derives from that claim.
