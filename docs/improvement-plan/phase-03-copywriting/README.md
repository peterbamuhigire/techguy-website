# Phase 3: Sales Copywriting Overhaul

## Score at Entry: 7.4/10 → Score at Exit: 7.7/10

Phase 2 established the positioning and category. Phase 3 makes that positioning audible across every page. Every headline is rewritten. The CTA ladder is built from scratch. Body copy is rewritten using the Sugarman slippery-slide principle so that every opening sentence earns the next. The contact page and services page receive benefit-led rewrites.

This phase is the highest-leverage copy investment in the entire plan. Headline rewrites alone have documented lift in engagement time, scroll depth, and enquiry rate.
