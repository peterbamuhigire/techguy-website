# Phase 6: Technical SEO and Structured Data

## Score at Entry: 8.3/10 → Score at Exit: 8.5/10

Phase 5 built the geographic pages. Phase 6 makes them — and every existing page — legible to search engines. This phase upgrades four categories of technical SEO: BlogPosting schema for all blog articles; FAQPage schema as the highest-ROI AI/LLM search signal; sitemap language handling to fix the `fr-FR` locale error in the Astro config; and page-by-page title and meta description audit against primary keyword assignments.

The phase also builds the Google Business Profile brief — the single highest-impact local SEO action available for a Kampala-based practice.
